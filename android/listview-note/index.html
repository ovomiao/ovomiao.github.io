Displays a vertically-scrollable collection of views, where each view is positioned immediately below the previous view in the list. -- 官方文档

【怎么用】：ListView的使用可以分为三步。第一步获取数据源，第二步实例化适配器，第三步为ListView设置适配器

1.ListView的XML属性

属性 说明
divider item分割线
dividerHeight 分割线高度
listSelector 点击效果

2.ListView常用方法

方法 说明
addFooterView(View v) ListView底部添加一个View
addHeaderView(View v) ListView头部添加一个View
setAdapter(ListAdapter adapter) 设置适配器
setDivider(Drawable divider) item分割线
setDividerHeight(int height) 分割线高度

3.ListView的使用

ListView需要通过适配器来加载数据。ListView有ArrayAdapter、SimpleAdapter、BaseAdapter 这三种适配器。ArrayAdapter和SimpleAdapter是BaseAdapter的直接子类。
